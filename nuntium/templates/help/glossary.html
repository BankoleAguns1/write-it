Glossary

This is a glossary of terms for WriteIt.

WriteIt is a zero-config factory for building sites that empower citizens to send emails to representatives and politicians.

How does WriteIt work?

Definitions

  • moderation
  • testing mode
moderation

If moderation is enabled, the site will not send messages until the site owner has approved them.

testing mode

A WriteIt site in testing mode will not send messages to the email addresses in its database — instead it will send them to the site's owner. Also, WriteIt doesn't publicise testing sites: whenever WriteIt displays a list of sites, testing sites are not shown (unless you're logged in as the sites' owner).
