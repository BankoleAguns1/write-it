Creating a WriteIt Instance

In order to use WriteIt you need to create an instance and provide a Popit Instance Api, for instance:

At Ciudadano Inteligente we have a Popit Instance of Chilean Parlamientarians which can be found at https://pmocl.popit.mysociety.org/ it's API is something similar https://pmocl.popit.mysociety.org/api/v0.1

Creating contacts for the people in your WriteIt Instance

Once you created an instance people should be related to it. Now you need to tell WriteIt how to reach that people, for that you need to create contacts for them. That can be done in here. You have to create a contact for each person.
